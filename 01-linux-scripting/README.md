# Lab 01: Linux Scripting

## Goal

Create a basic Linux system information script using Bash.

## Script

`system-info.sh`

## What It Shows

The script prints:

- hostname
- current user
- current date
- system uptime
- disk usage
- memory usage
- top 5 processes by memory usage

## Commands Practiced

```bash
pwd
ls
cd
touch
chmod +x system-info.sh
./system-info.sh
df -h
free -h
ps aux
